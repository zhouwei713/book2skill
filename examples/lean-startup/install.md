# Install Guide: The Lean Startup Skill Pack

Copy each folder under `skills/` into your Agent skills directory.

Example:

```bash
cp -R skills/* ~/.claude/skills/
```

Then ask your Agent to apply a skill by describing the task.

Examples:

```text
Use the MVP skill to design a minimum viable product for my new AI tool idea.
```

```text
Use the Pivot skill to decide whether my current product direction should change.
```
