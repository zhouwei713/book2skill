# Skill Generation Prompt

## Task

Generate one installable `SKILL.md` file for each approved methodology.

## Requirements

1. Use clear and precise frontmatter.
2. The `description` must explain when the skill should trigger.
3. Each skill must have when-to-use guidance, inputs, process, output format, quality rules, and source note.
4. Keep the method actionable.
5. Do not copy long passages from the source material or from external sources.
6. Mark confidence level clearly.
7. Use lowercase kebab-case for folder names.

## Folder output

```text
skills/
  {skill_name}/
    SKILL.md
```
