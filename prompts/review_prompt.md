# Quality Review Prompt

## Task

Review the generated methodology list and generated skills before delivery.

## Checklist

1. Each skill has a clear trigger.
2. Each skill has a unique name.
3. Each skill has a practical use case.
4. Each skill has required inputs.
5. Each skill has step-by-step execution.
6. Each skill has a fixed output format.
7. Skills do not duplicate one another.
8. The source basis is recorded.
9. Copyright-sensitive content is summarized instead of copied.
10. Uncertainty is clearly marked.

## Output

Create `quality-report.md` with pass, warn, or fail for each skill.
