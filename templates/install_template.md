# Install Guide: {{book_title}}

## Contents

This package contains generated skills from {{book_title}} by {{author}}.

## Install as folders

Copy each folder under `skills/` into your Agent skills directory.

Example:

```bash
cp -R skills/* ~/.claude/skills/
```

## Install as packaged files

If your platform supports packaged skill files, use the zip files in `dist/`.

## Suggested usage

After installation, ask your Agent to use a specific skill by describing the task or mentioning the skill name.

Examples:

```text
Use the MVP skill to help me design a minimum viable product for this idea.
```

```text
Use the Pivot skill to diagnose whether my product needs a strategic shift.
```
